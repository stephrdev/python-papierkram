Requirements
------------

python-papierkram supports Python 3 only.


Prepare for development
-----------------------

A Python 3.7 interpreter is required in addition to pipenv.

.. code-block:: shell

    $ pipenv install --python 3.7 --dev
    $ pipenv shell
    $ pip install -e .


Now you're ready to run the tests:

.. code-block:: shell

    $ pipenv run py.test
